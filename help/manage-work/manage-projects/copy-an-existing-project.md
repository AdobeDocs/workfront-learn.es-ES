# Copiar un proyecto existente de forma eficaz

En el vídeo se explica cómo copiar un proyecto en lugar de utilizar una plantilla. Para copiar un proyecto, abra el proyecto deseado, haga clic en el menú de tres puntos junto al nombre del proyecto y seleccione &quot;Copiar&quot;. En la ventana Copiar proyecto, puede cambiar el nombre del proyecto y su estado, normalmente a &quot;Planificación&quot; para evitar enviar notificaciones a los usuarios asignados. Puede elegir si desea copiar asignaciones, progreso, datos personalizados, etc.
Para finalizar, haga clic en &quot;Copiar proyecto&quot;. &#x200B;
Los problemas no se pueden copiar directamente durante este proceso, ya que no se consideran trabajo planificado. Para copiar problemas, vaya a la pestaña Problemas, seleccione los problemas que desee y utilice la opción Copiar en para transferirlos al nuevo proyecto. &#x200B;


>[!VIDEO](https://video.tv.adobe.com/v/3456037/?quality=12&learn=on&enablevpops=1)

## Principales conclusiones

* **Copiando un proyecto:** Abra el proyecto, haga clic en el menú de tres puntos situado junto al nombre del proyecto y seleccione &quot;Copiar&quot; para duplicarlo. &#x200B;
* **Ajuste de la configuración del proyecto:** Cambie el nombre del proyecto y su estado a &quot;Planificación&quot; para evitar enviar notificaciones a los usuarios asignados.
* **Copia selectiva:** Puede elegir si desea copiar asignaciones, progreso o datos personalizados. Anular la selección de estas opciones permite la personalización en el nuevo proyecto.
* **Administrar datos personalizados:** Si no se seleccionan los datos personalizados, los formularios se copiarán sin sus datos. Deje la casilla marcada para copiar ambos formularios y sus datos. &#x200B;
* **Copiar problemas por separado:** los problemas no se copian durante el proceso de duplicación del proyecto. Para copiarlos, vaya a la pestaña Problemas, seleccione los problemas deseados y utilice la opción Copiar en para transferirlos al nuevo proyecto. &#x200B;
