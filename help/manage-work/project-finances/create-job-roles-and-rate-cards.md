# Creación de puestos y tarjetas de tarifas

En este vídeo se explica cómo crear un rol y una tarjeta de tarifa, y cómo adjuntar y quitar una tarjeta de tarifa de un proyecto.


>[!VIDEO](https://video.tv.adobe.com/v/3486521/?captions=spa&quality=12&learn=on&enablevpops=1)

## Principales conclusiones

* **Funciones del puesto Estructura de tareas del proyecto:** Las funciones del puesto representan las habilidades necesarias para las tareas y se pueden asignar a los usuarios en función de sus capacidades y disponibilidad.
* **Tarjetas de tarifas Defina las tarifas de facturación:** Las tarjetas de tarifas enumeran las tarifas de facturación para los roles de trabajo y actúan como contratos con los clientes para la facturación del proyecto.
* **Administración flexible de tarifas:** Tanto los roles de trabajo como las tarjetas de tarifas permiten establecer tarifas de costo y facturación con fechas efectivas, lo que admite cambios de precios basados en el tiempo.
* **Anulaciones de tarjeta de tarifa:** Cuando se adjunta una tarjeta de tarifa a un proyecto, sus tarifas anulan las tarifas de funciones del puesto, pero las invalidaciones específicas del proyecto no se ven afectadas.
* **Actualizaciones y archivos adjuntos sencillos:** Las tarjetas de tarifas se pueden adjuntar o eliminar de los proyectos, y las actualizaciones se pueden insertar en proyectos existentes para garantizar una facturación precisa.
